ETL de Ventas en winform con C# y SQLServer 

-- Descripción --

Este proyecto consiste en el desarrollo y análisis de un proceso ETL (Extract, Transform, Load) en C#, que permite procesar información de ventas y almacenarla en una base de datos.

Los únicos recursos proporcionados fueron tres archivos de datos:

ventas_fake.json

ventas_fake.xml

ventas_fake.txt

A partir de estos archivos, desarrollé toda la solución necesaria para leerlos, transformar la información y cargarla correctamente en la base de datos.

-- ¿Qué hace el sistema? --

Permite seleccionar archivos en distintos formatos.

Procesa y unifica la información en un modelo común.

Elimina registros duplicados.

Guarda los datos de forma estructurada en la base de datos.

-- Objetivo --

Demostrar la capacidad de analizar y desarrollar un proceso ETL completo en C#, partiendo únicamente de archivos de datos y construyendo toda la lógica necesaria para su procesamiento y almacenamiento.
